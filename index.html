Q1 ------> 5 meta tag and there uses:

  1. (meta charset="UTF-8")-----> The character encoding, emojis create
  2. (meta name="keywords" content=" ")-----> Short or mid description about the content keywords of our website
  3. (meta name="description" content=" ")-----> Short or mid description about the contents of our website
  4. (meta name="author" content=" ")-----> The name of the author of the webpage
  5. (meta name="refresh" content="50")-----> writing this code will make sure the browser refreshes after every 50seconds
